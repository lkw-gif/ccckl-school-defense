# 基朗突擊：校園防線

原創 16-bit 橫向校園槍戰遊戲。玩家需要擊破入侵校園的怪物、拾取武器、跨越障礙和跳台，依次清除校門、科學樓、課室及天台戰區。

## 操作

- `A` / `D`：左右移動
- `SPACE`：跳躍
- `W` / `↑`：向上瞄準
- `S` / `↓`：趴下；空中時向下瞄準
- `J` / `Z` / `X`：射擊
- `SHIFT` / `K`：衝刺
- `E` / `C`：切換武器
- `Q`：手榴彈
- `M`：切換音樂

手機和平板可使用畫面上的觸控按鈕遊玩。

## 本機執行

遊戲是純靜態網站，可直接使用任何本機 HTTP server：

```bash
python3 -m http.server 4173
```

然後開啟 <http://127.0.0.1:4173/>。
